# Cyber-Gaurd-
Cyber Gaurd is a program built on the principle of Proactive Digital Defense. In a market saturated with reactive tools, this program positions itself as a comprehensive guardian that integrates human intelligence with automated precision, the name serves as a brand promise to maintain the CIA triad (Confidentiality, Integrity, and Availability).
